## Prerequsite

- NodeJS >= v12.18.3

## Installation

1. clone repo

```
$ cd teddy-nft
```

2. copy `.env.example` file & rename it to `.env`

3. add relevant data as mentioned in it

4. install node modules

```
$ npm i
```

5. start web server

```
$ npm start
```

6. server will start on port define in `.env` or `3000`(default)
